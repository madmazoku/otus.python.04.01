Usage
=====

Options

.. code-block:: 

    ./api.py [-p PORT] [-l LOG_FILE_NAME]

        score service server

        -p : port to be listened, default 8080

        -l : log file name, default print to stderr


Example
-------

Execute script:

.. code-block:: 

    ./api.py

Execute tests:

.. code-block:: 

    ./test.py
